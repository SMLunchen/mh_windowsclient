Meshhessen Client

Kostenloser Windows-Client für Meshtastic-Geräte – Offline-Karte, Telemetrie, Traceroute und PKI-Entschlüsselung. Keine Cloud, keine Installation.

Download (EXE) GitHub Repository meshhessen.de

Free Windows app for Meshtastic devices – offline map, telemetry, traceroute, PKI decryption, USB/Serial/TCP/BLE. No installation, no cloud. By the Meshhessen community (Hesse, Germany).

Features

📨 Nachrichten

Broadcast & Direktnachrichten, Multi-Channel, Tap-Back Reaktionen, Alert Bell / SOS-Notruf.

🗺️ Offline-Karte

OSM Standard, OSM Dark, OpenTopoMap – Tiles für ganz Deutschland offline speicherbar. Node-Pins, GPS-Tracks.

📊 Telemetrie

SQLite-Datenbank, Zeitreihen-Graphen (SNR, RSSI, Batterie), LED-Ampel für Mesh-Health.

📡 Traceroute

Hop-Tabelle mit Entfernung & SNR, Karten-Visualisierung, mehrere Traces gleichzeitig, JSON-Speicherung.

🔒 PKI-Entschlüsselung

Client-seitige X25519 + AES-256-CTR Entschlüsselung. Private Key wird automatisch vom Gerät geladen.

⚙️ Verbindung

USB/Serial, TCP/WiFi, Bluetooth BLE – Auto-Reconnect, Update-Check, Dark Mode, Deutsch/Englisch.

Unterstützte Geräte

Funktioniert mit allen Meshtastic-kompatiblen Geräten: LILYGO T-Beam · LILYGO T-Deck · RAK4631 · Heltec LoRa 32 · Heltec Mesh Node · Seeed WiO-E5 · und weitere. Verbindung per USB/Serial (115200 baud), TCP/WiFi oder Bluetooth Low Energy (BLE).

Download & Installation

  1. .NET 8 Desktop Runtime herunterladen und installieren
  2. Neueste MeshhessenClient.exe herunterladen
  3. Meshtastic-Gerät per USB anstecken
  4. Doppelklick auf MeshhessenClient.exe – fertig. Keine Installation nötig.
