Demnächst Persistente Nachrichten-DB & Verbindung merken

Nachrichten überleben Neustarts. Die App merkt sich, womit du zuletzt verbunden warst.

✨ Hinzugefügt

Persistente Nachrichten-Datenbank
  • SQLite-Nachrichtenspeicher für Kanal- und DM-Nachrichten — optional, in den Einstellungen aktivierbar
  • Je Kanal eine eigene DB-Datei (messages/channel_{index}_{name}.db), DMs in messages/dm.db
  • WAL-Modus, Insert/LoadSince/LoadBefore/ClearAll/ClearOlderThan, filterbar per partner_id
  • Automatisches Laden der letzten 24 h nach dem Verbinden; ältere Nachrichten per Hochscrollen nachladen
  • DM-History: beim Öffnen des DM-Fensters werden alle gespeicherten Konversationen automatisch als Tabs wiederhergestellt
  • Pro-Kanal-Löschung im Kanäle-Tab — neuer Button pro Zeile, öffnet Dialog mit Zeitraum-Auswahl (Alle / 30 / 90 / 365 Tage)
  • Bestätigung vor dem Leeren einer DM-Konversation
  • Aufbewahrungsdauer konfigurierbar, Retention wird beim Start angewendet
  • Neue Einstellungen: EnableMessageDb, MessageDbRetentionDays
Verbindung merken
  • Letzte Verbindungsart (Serial / Bluetooth / WiFi) wird gespeichert und beim Start automatisch vorausgewählt
  • Letztes BT-Gerät wird gespeichert; nach dem BT-Scan automatisch in der Liste vorausgewählt
  • Neue Einstellungen: LastConnectionType, LastBtDevice
v1.5.8 Online-Karten, HTTP/3 & neuer Tile-Server

Drei Kartenmodi, eigener Tile-Server, HTTP/3-Support und OSM-konformes Caching.

✨ Hinzugefügt

Karten-Erweiterungen
  • Online-Karten-Modi: Meshhessen-Server (lazy download, Deutschland) und OpenStreetMap (weltweit, Policy-konform)
  • Neuer Tile-Server: tile.meshhessenclient.de löst den bisherigen Server ab
  • HTTP/3 mit automatischem Fallback auf HTTP/2 und HTTP/1.1
  • OSM-Tile-Cache mit HTTP-Cache-Headers (ETag/If-None-Match, min. 7 Tage TTL)
  • OSM Dark Mode und OpenTopoMap als neue Kartentypen
  • Tile-Server-URL in Einstellungen konfigurierbar
  • Copyright-Hinweise auf der Karte, dynamisch je nach Quelle
Alert Bell / SOS
  • SOS-Button in Hauptchat und DM-Fenster
  • Visuelle Benachrichtigung: rote blinkende Umrandung (6 Blinks, 3 Sekunden)
  • Akustische Benachrichtigung (WAV-generiert, unabhängig von System-Stummschaltung)
  • Notification-Bar mit Absender und „Zur Karte springen"-Button
  • Verschwindet automatisch nach 30 Sekunden oder manuell schließbar

🐛 Behoben

  • Zombie-Prozess: App beendet sich jetzt sauber (Application.Current.Shutdown())
  • Tab-Navigation: „Zur Karte"-Button springt korrekt zum Karten-Tab

🔄 Geändert

  • Alert Bell Format: von ASCII 0x07 auf Emoji 🔔 umgestellt — kompatibel mit Android & Web

v1.5.8 Online Maps, HTTP/3 & New Tile Server

Three map modes, own tile server, HTTP/3 support and OSM-compliant caching.

✨ Added

Map Enhancements
  • Online map modes: Meshhessen server (lazy download, Germany) and OpenStreetMap (worldwide, policy-compliant)
  • New tile server: tile.meshhessenclient.de replaces the previous server
  • HTTP/3 with automatic fallback to HTTP/2 and HTTP/1.1
  • OSM tile cache with HTTP cache headers (ETag/If-None-Match, min. 7-day TTL)
  • OSM Dark mode and OpenTopoMap as new map types
  • Tile server URL configurable in settings
  • Dynamic copyright attribution on map
Alert Bell / SOS
  • SOS button in main chat and DM window
  • Visual alert: red flashing border (6 blinks over 3 seconds)
  • Audio alert (WAV-generated, works even when system is muted)
  • Notification bar with sender name and "Jump to map" button
  • Auto-dismisses after 30 seconds or manually closeable

🐛 Fixed

  • Zombie process on exit: app now shuts down cleanly via Application.Current.Shutdown()
  • Tab navigation: "Go to map" button now correctly activates the Map tab

🔄 Changed

  • Alert Bell format: switched from ASCII 0x07 to emoji 🔔 — compatible with Android & web apps
v1.0-Beta 2026-02-08 Initial Release

First public beta with DMs, message logging and full telemetry analysis.

✨ Added

Direct Messages (DMs)
  • Dedicated DM window with tab system for multiple conversations
  • Automatic notifications (foreground, taskbar flash, system sound)
  • Orange highlight and bold text for unread messages
  • "💬 Send DM" via right-click on node
Message Logging
  • Automatic logging of all messages to separate log files
  • Separate files per channel (logs/Channel_[Index]_[Name].log) and per DM partner
  • UTF-8 encoding, thread-safe writing
Map & Telemetry
  • Offline OSM map with node pins, GPS tracks, waypoints
  • Telemetry analysis: SNR, RSSI, battery, channel utilization, routing, neighbors
  • Traceroute with hop table and map visualization
  • PKI decryption (X25519 + AES-256-CTR, key loaded from device)
  • LED indicators for signal, neighbors, path, mesh health, weather
Connection & System
  • USB/Serial, TCP/WiFi, Bluetooth BLE — auto-reconnect
  • Tap-back reactions (32 emojis), quote-block replies
  • Multi-channel support, firmware version & hardware model display
  • Update check, dark mode, German/English
